# weather-tracker
This is a AI agent that checks the weather of Inks Lake State Park



I built a project that will check the weather in July at Inks Lake State Park over the years to see if it is good to go camping this year. This compares the historical July weather with a live 7-day forecast. The data it uses comes from the  Open-Meteo API archive, which we will use to turn into a CSV file.
